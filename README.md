# NEBRAS – لوحة معلومات المنزل الذكية

NEBRAS هو نموذج أولي لحل ميداني ذكي يعتمد على شريحة NFC تُثبت عند مدخل المنزل،
ليتيح لرجال الأمن والدفاع المدني الوصول السريع إلى بيانات أساسية عن المنزل
أثناء الطوارئ، مثل حالة السكن، عدد الأفراد، وجود أطفال، الحالات الصحية،
والمواد الخطرة.

## فكرة المشروع

عند تمرير جهاز الجوال على شريحة الـ NFC المثبّتة على الباب، يتم فتح صفحة ويب
خاصة بكل منزل تعرض:

- حالة المنزل (مأهول / غير مأهول)  
- عدد الأفراد  
- وجود أطفال  
- الحالة الصحية الخاصة  
- المواد الخطرة (مثل أسطوانة الغاز)  
- رقم التواصل السريع  
- معرف المنزل (Home ID) مثل: `NB-0001`  
- تاريخ آخر تحديث للبيانات  

كما تحتوي الواجهة على زر **"مباشرة البلاغ"** الذي ينقل المستجيب الميداني إلى صفحة
تأكيد تفيد بأنه تم مباشرة البلاغ لهذا المنزل.

## المكوّنات الأساسية

- `index.html` – صفحة عرض بيانات المنزل (القراءة)  
- `report.html` – صفحة تأكيد مباشرة البلاغ  
- `nebras-logo.png` – شعار NEBRAS المستخدم في الواجهة  

## طريقة التشغيل

1. فتح رابط GitHub Pages للمشروع:  
   `https://saraalmamoud.github.io/nebras-interface/`
2. ربط الرابط بشريحة NFC باستخدام تطبيق مثل **NFC Tools**.
3. عند تمرير جهاز رجل الأمن على الشريحة، تفتح صفحة NEBRAS تلقائيًا.

## التقنيات المستخدمة

- HTML  
- CSS  
- (اختياري لاحقًا) JavaScript ودمج بيانات ديناميكية

## فريق العمل

فريق **نبراس** – خريجو نظم معلومات، بخبرة في دعم وتطوير التطبيقات
وتصميم حلول رقمية ميدانية تعزز السلامة المجتمعية.
